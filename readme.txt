-------------------------------------------------
-                Quantum Arena                  -
-------------------------------------------------

A Ludum Dare 49 by Aumniez & Iarwain.

Multiplayer versus game, from 2 to 4 players.

Frank Brizuela
Romain Killian

-------------------------------------------------

Controls:

Player1
-------
Move                    - WASD
Attack                  - Arrow keys
            .. or ..
Gamepad #1
Move                    - Left Stick or D-Pad
Attack                  - ABXY

Player2
-------
Gamepad #2

Player3
-------
Gamepad #3

Player4
-------
Gamepad #4

Menu
----
Key 2, 3 or 4           - Select Player(s) Count
            .. or ..
A/Start on Gamepad #1   - 2P
A/Start on Gamepad #2   - 2P
A/Start on Gamepad #3   - 3P
A/Start on Gamepad #4   - 4P

Game Over
---------
Space/Start/Back        - Go back to Select Menu

Misc
----
F12 / Thumb Stick       - Capture a screenshot
